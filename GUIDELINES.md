# Guidelines
## Litros lee esto!!
- Cada caracerística/módulo nuevo tienes que desarrollarlo en un `branch` exclusivo:

```bash
# Para crear un branch:
git checkout -b mi_branch_nuevo
# Programai y commiteai igual que siempre:
git add .
git commit -m "[mi_branch_nuevo] Mensaje corto y preciso, con mayúsculas al principio y sin punto final, importante"
# Cuando estai listo vuelves al branch master:
git checkout master
# Y haces un merge de lo que programaste:
git merge mi_branch_nuevo
git push
```

- Eso. Cualquier cosa pregunta.
